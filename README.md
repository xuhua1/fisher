## 全局安装pipenv
1. pip3 install pipenv
## 进入项目
2. cd fisher
## 创建虚拟环境
3. pipenv install 
## 进入虚拟环境
4. pipenv shell
## 安装flask
5. pipenv install flask

## 安装包列表
* pip list
## 退出虚拟环境
* exit
## 卸载
* pipenv un install flask
## 查看安装包关系
* pipenv graph
